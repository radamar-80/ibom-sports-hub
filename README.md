Ibom Sports Hub

An e-commerce website for selling football boots, jerseys, and gear.

Features

- Product collections (Boots, Jerseys, Gear)
- Boot filtering (FG, AG, SG)
- Shop by brand (Nike, Adidas, Puma)
- Admin dashboard (add/edit products)
- Cart system
- Paystack payment integration
- WhatsApp order option

Tech Stack

- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB

Project Structure

/client → Frontend (React)
/server → Backend (API & logic)

How to Run

1. Install dependencies:
   npm install

2. Start server:
   node server.js

3. Run frontend:
   npm start
